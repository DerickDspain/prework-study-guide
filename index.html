HTML

  • HTML-Hypertext Markup Language.
  • The head element contains information about the webpage.
  • The Body element represents the visible content shown to the user.
  • Alt is used to include alternate text when the image can't be displayed.
  • div element is used to define a section.
  • href is an attribute of "a" it specifies a links destination.
  • src tells us where the image is coming from.
  • Nesting is when one element is placed inside another.

CSS

  • A margin indidcates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • git-status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and swithcnes to it.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow iss the order in which a computer executes code in a script.
  • An array allows us to contain more than one item to be called upon in our code.
